Typography

Type face

We use Arial in both regular and bold formats.

Typographic scale

Although we have a limited set of font sizes we are not dictating font sizes for HTML elements (e.g. headers, body copy, etc...). A flexible approach gives designers freedom to define different typographic hierarchies for the different type of pages and components we have across the product.
